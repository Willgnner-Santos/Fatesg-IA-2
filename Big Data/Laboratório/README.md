# Dashboard de Análise de Vendas da BMW (2010 - 2024)

Este projeto apresenta um dashboard interativo construído em Power BI para analisar e visualizar dados de vendas de veículos. O objetivo é extrair insights valiosos sobre o desempenho de vendas, identificando tendências, padrões e a influência de diferentes atributos nos resultados.


# Perguntas Respondidas:

O dashboard foi desenvolvido para responder a perguntas-chave de negócio, fornecendo uma visão clara e objetiva do cenário de vendas. As principais análises incluem:

Modelo Mais Vendido por Ano: Identificação do modelo de veículo com o maior volume de vendas em cada ano.

Análise de Vendas por Tipo de Combustível: Avaliação da performance de vendas de veículos a gasolina, elétricos, híbridos e a diesel ao longo do tempo.

Influência da Transmissão nas Vendas: Uma comparação entre o volume de vendas de veículos com transmissão manual versus automática.

Relação entre Quilometragem e Preço: Análise da correlação entre a quilometragem e o preço médio de venda, utilizando um gráfico de dispersão para visualizar o impacto.

Preço Médio por Modelo: Determinação do preço médio de venda para cada modelo de veículo.


# Tecnologias Utilizadas:

Power BI: Principal ferramenta para modelagem de dados, criação de medidas DAX e visualização interativa.

DAX (Data Analysis Expressions): Linguagem utilizada para criar medidas e colunas calculadas, permitindo análises aprofundadas como Vendas Totais e Preço Médio.


# Como Visualizar o Dashboard

1. Clone este repositório para a sua máquina local.

2. Abra o arquivo .pbix no Power BI Desktop.

3. Interaja com os filtros e visuais para explorar os dados e obter seus próprios insights.
