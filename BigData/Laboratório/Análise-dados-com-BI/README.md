# Análise de Dados de Vendas BMW com Power BI

Este projeto documenta o processo de análise de dados de vendas da BMW utilizando o **Microsoft Power BI**. O objetivo foi transformar um conjunto de dados brutos (`BMW sales data (2010-2024).csv`) em insights visuais claros e compreensíveis. Este exercício me permitiu aprofundar meu conhecimento em modelagem de dados, criação de dashboards interativos e storytelling com dados.

## Metodologia e Execução

O trabalho foi conduzido inteiramente no ambiente do Power BI Desktop. As etapas seguiram uma progressão lógica, refletindo um fluxo de trabalho padrão de Business Intelligence, desde a ingestão dos dados até a visualização final no dashboard.

### 1. Ingestão e Preparação de Dados

O primeiro passo foi importar o arquivo CSV, que continha dados de vendas da BMW entre 2010 e 2024. Usei a função "Obter Dados" para carregar o arquivo no Power BI.

### 2. Modelagem de Dados

Não houve a necessidade de criar um modelo de dados complexo, pois os dados já estavam contidos em uma única tabela. No entanto, este projeto me serviu para praticar a exploração e a validação do conjunto de dados, garantindo que as colunas estavam com os tipos de dados corretos e que não havia inconsistências.

### 3. Análise e Visualização

A etapa mais importante foi a criação das visualizações no dashboard. As imagens de evidência mostram a aplicação de diferentes gráficos e elementos visuais para contar uma história com os dados:

-   **Dashboard Principal (`Evidencia01 .png`)**: A imagem mostra um dashboard com vários elementos visuais para fornecer uma visão completa das vendas.
    -   **Gráfico de Vendas por Ano**: Um gráfico de colunas que mostra a evolução das vendas ao longo do tempo, destacando a variação ano a ano.
    -   **Gráfico de Vendas por Modelo**: Um gráfico de barras que compara o desempenho de vendas dos diferentes modelos da BMW.
    -   **Indicadores de Desempenho (KPIs)**: Cartões que exibem métricas importantes, como o total de vendas.
    -   **Slicers (Filtros)**: Filtros por "Ano" e "Modelo" foram adicionados para permitir uma análise interativa, onde o usuário pode selecionar um ano ou modelo específico e ver como os outros gráficos reagem.

### 4. Conclusão

Este exercício me proporcionou um conhecimento prático e valioso sobre como transformar dados brutos em inteligência de negócios. Pude aplicar conceitos de ETL (Extract, Transform, Load) e visualização de dados para criar um dashboard funcional e informativo. A experiência de construir um painel interativo no Power BI reforçou minha compreensão sobre a importância de apresentar os dados de forma clara e acessível, capacitando a tomada de decisões estratégicas.
