# Projeto de Análise e Limpeza de Dados com Pandas

Este projeto consiste em um estudo prático de manipulação, exploração e limpeza de dados utilizando a biblioteca Pandas em Python. O trabalho foi desenvolvido como parte do "Kit Prático – Semana 2" sobre manipulação de dados.

## Objetivo Principal

O objetivo é aplicar técnicas fundamentais de ciência de dados para analisar o conjunto de dados do Titanic. As principais tarefas incluem:
- Carregamento e visualização inicial dos dados.
- Exploração das características (colunas) do dataset.
- Filtragem de dados para extrair informações específicas (e.g., sobreviventes, passageiros por idade e sexo).
- Limpeza de dados, tratando valores ausentes e duplicados.
- Análise contextual, como a investigação de possíveis fraudes de seguro a partir dos dados.

## Estrutura do Projeto

O repositório contém os seguintes arquivos:

- **`KitSemana2_final.ipynb`**: Um Jupyter Notebook que contém todo o código Python e a análise passo a passo do dataset do Titanic. Inclui a importação de dados, visualização, filtragem e as etapas de limpeza.

- **`diarias_concatenadas.csv`**: Um segundo conjunto de dados, proposto como um desafio para que o aluno aplique os mesmos conceitos aprendidos com o dataset do Titanic.

- **`Evidencia 01.png` a `Evidencia 11.png`**: Imagens (screenshots) que servem como evidência da execução e dos resultados obtidos no notebook.

## Como Utilizar

1. Abra o arquivo `KitSemana2_final.ipynb` em um ambiente que suporte Jupyter Notebooks (como Jupyter Lab, VS Code ou Google Colab).
2. Execute as células de código sequencialmente para acompanhar a análise de dados do Titanic.
3. Para o desafio, utilize o conhecimento adquirido para analisar o arquivo `diarias_concatenadas.csv`.

## Ferramentas Utilizadas

- **Python**
- **Pandas**: Para manipulação e análise de dados.
- **Jupyter Notebook**: Para a execução interativa do código.
